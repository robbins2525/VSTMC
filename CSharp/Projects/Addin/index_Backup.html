Update

Using User Environment variable settings to store MicroStation CONNECT Edition settings. With new Visual Studio load requirements, creating Visual Studio macros or internal environment variables is unreliable. Go to Visual Studio Tools|Options… and select the Bentley|MicroStation CONNECT leaf. Verify your settings and click OK. If the settings are not currently stored in the User Environment, the settings will be saved to the User Environment. You need to restart Visual Studio for changes to take effect.
